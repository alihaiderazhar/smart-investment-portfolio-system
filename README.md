# Smart Investment Portfolio Management System (SIPMS)

A Python-based financial management program built to bridge theoretical 
OOP concepts with real-world FinTech applications.

## Overview
SIPMS allows users to manage, analyze, and evaluate financial investments 
across multiple asset classes including **Stocks**, **Mutual Funds**, and 
**Cryptocurrency** — with a focus on clean, structured code design.

## Key Technical Highlights

- **Inheritance** — Asset classes share a common base structure
- **Polymorphism** — Different asset types handled through unified interfaces
- **Abstract Classes** — Enforced structure across all investment types

## Built With

- Python 3
- NumPy
